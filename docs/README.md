### 📚 [Yahoo Finance 한국 주식 연동 문제 해결 가이드](./YAHOO_FINANCE_TROUBLESHOOTING.md)

**언제 참고하나요?**
- 포트폴리오 수익률이 0.00%로 표시될 때
- Yahoo Finance API 호출 시 오류가 발생할 때
- 한국 주식(KOSPI/KOSDAQ) 데이터를 가져오지 못할 때
- yahoo-finance2 라이브러리 업데이트 후 문제가 발생했을 때

**주요 내용**:
- yahoo-finance2 v3의 올바른 사용법
- 한국 주식 심볼 형식 (.KS, .KQ)
- 문제 진단 및 해결 방법
- 테스트 스크립트 예제
