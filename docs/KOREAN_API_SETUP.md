# 한국 주식 API 키 신청 가이드

## 1. DART API 키 (즉시 발급)

### 신청 절차
1. **OPEN DART 포털 접속**
   - https://opendart.fss.or.kr/

2. **회원가입 및 로그인**
   - 우측 상단 "회원가입" 클릭
   - 개인 또는 기업 회원가입 진행

3. **API 키 신청**
   - 로그인 후 상단 메뉴 **"오픈API"** 클릭
   - "인증키 신청/관리" 선택
   - "인증키 신청" 버튼 클릭
   - 서비스명 입력 (예: "개인 포트폴리오 관리")
   - **즉시 발급됨**

4. **API 키 확인**
   - "오픈API 이용현황" 페이지에서 발급된 키 확인
   - **API 키를 복사해두세요** (40자리 영숫자)

### 예상 시간: **5-10분**

## 2. KRX API 키 (승인 필요 ~1영업일)

### 신청 절차
1. **KRX OPEN API 시스템 접속**
   - https://data.krx.co.kr/

2. **회원가입**
   - 우측 상단 "회원가입" 클릭
   - 개인 회원가입 진행 (이메일 인증 필요)

3. **로그인 후 API 인증키 신청**
   - 로그인 → "마이페이지" 클릭
   - "Open API" 탭 선택
   - "API 인증키 신청" 클릭
   - 이용목적 입력 (예: "개인 투자 포트폴리오 관리")
   - 신청 완료 → **승인 대기 (~1영업일)**

4. **API 서비스 신청 (인증키 승인 후)**
   - 마이페이지 → "서비스 이용" 메뉴
   - 다음 **6개 항목** 신청:
     1. KOSPI 일별 매매정보
     2. KOSPI 기본정보
     3. KOSDAQ 일별 매매정보
     4. KOSDAQ 기본정보
     5. KONEX 일별 매매정보
     6. KONEX 기본정보
   - 각 항목마다 "신청" 버튼 클릭 → 승인 대기

5. **API 키 확인**
   - 마이페이지에서 승인된 API 키 확인
   - **API 키를 복사해두세요**

### 예상 시간: **신청 10분 + 승인 대기 ~1영업일**

## 3. API 키 설정

### .env.local 파일 업데이트

API 키를 받으면 `.env.local` 파일에 추가하세요:

```bash
# 기존 API 키들...
FINNHUB_API_KEY=d5vdql...
GEMINI_API_KEY=AIzaSy...

# 한국 주식 API 키 (새로 추가)
DART_API_KEY=<여기에_DART_API_키_입력>
KRX_API_KEY=<여기에_KRX_API_키_입력>
```

### 주의사항
- API 키는 절대 Git에 커밋하지 마세요
- `.env.local`은 이미 `.gitignore`에 포함되어 있습니다
- API 키는 외부에 노출되지 않도록 주의하세요
